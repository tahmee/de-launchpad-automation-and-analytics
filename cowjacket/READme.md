# Jira Ticket Automation 

An automated Python-based system that synchronizes phone equipment requests from a database to Jira Service Desk, creating tickets with intelligent duplicate detection and comprehensive error handling.

## Navigation / Quick Access

- [Overview](#overview)

## Overview

This system automates the creation of Jira Service Desk tickets for phone equipment requests. It reads requests from a PostgreSQL database, processes them in batches, and creates corresponding Jira tickets with all necessary details including equipment types, installation requirements, and timeframes.
